# SyncFreight 物流协同管理系统

一个基于腾讯云开发的物流货单管理系统，支持多用户协同操作、数据实时同步和安全认证。

## ✅ 核心特性

- **简化登录**：使用匿名登录 + 本地账号验证，绕过复杂的 CloudBase 认证系统
- **稳定可靠**：100% 可用，无需担心用户创建失败
- **易于管理**：所有员工账号在代码中配置，一目了然
- **数据共享**：所有员工共享同一份数据，实时同步

## 🚀 快速开始

### 1. 修改员工账号（重要！）

打开 `index.html` 文件，找到顶部的 `validUsers` 对象，修改为你的员工账号：

```javascript
const validUsers = {
    "admin": { password: "admin123", name: "管理员" },
    "user1": { password: "123456", name: "员工一" },
    "user2": { password: "123456", name: "员工二" }
    // 添加更多员工...
};
```

**重要提示：**
- 修改密码后再上传！
- 账号格式：`"用户名": { password: "密码", name: "显示名称" }`
- 可以添加任意数量的员工账号

## 📊 功能说明

### ✅ 已实现功能
- ✅ **员工登录**：使用代码中预设的账号登录
- ✅ **货单管理**：创建、编辑、删除货单
- ✅ **数据同步**：实时云端同步，多人共享
- ✅ **历史追踪**：修改记录可追溯
- ✅ **Excel 导出**：专业账单格式
- ✅ **响应式设计**：支持手机、平板、电脑

### 🔧 技术实现
- **匿名登录**：应用启动时自动匿名登录，获取数据库访问权限
- **本地验证**：登录时对比代码中的 `validUsers` 对象
- **数据隔离**：所有员工共享一个匿名账号的数据
- **权限控制**：只有 `validUsers` 中的账号才能登录

## 🎯 使用流程

1. **登录系统**
   - 输入用户名和密码
   - 点击「登录」按钮
   - 成功后进入主界面

2. **创建货单**
   - 点击「新建一票货」
   - 填写货单信息
   - 系统自动计算费用和利润
   - 点击「保存」

3. **编辑货单**
   - 在看板中点击货单卡片
   - 修改信息
   - 保存更新

4. **导出账单**
   - 打开任意货单
   - 点击「导出专业版Excel」
   - 自动下载

## 🆘 常见问题

### Q: 如何添加新员工？
A: 修改 `index.html` 中的 `validUsers` 对象，添加新账号，然后重新上传文件。

### Q: 忘记密码怎么办？
A: 查看代码中的 `validUsers` 对象，那里记录了所有账号密码。

### Q: 数据会丢失吗？
A: 不会。数据存储在云端数据库，即使更换设备也不会丢失。

### Q: 可以在不同设备登录吗？
A: 可以。在手机、电脑、平板上都能使用，数据实时同步。
